# CMPG313_Lab1_YOLOv8
Training a Machine Learning model using YOLOv8 to recognize and track objects via image, video, and live webcam feed.

Object Detection and Tracking with YOLOv8Project OverviewThis repository contains the practical implementation for Lab 1: 
Training a Machine Learning Model with YOLOv8. The objective of this lab was to utilize a pre-trained YOLOv8 model to recognize and track objects—including myself—using images, video files, and a live webcam feed.
Learning Outcomes
Gained hands-on experience with Python scripting and the Ultralytics library.
Explored the differences between object detection (predict) and object tracking (track).
Understood model confidence behavior and the limitations of computer vision in real-time scenarios.
